# SptecLazyLoad

Simple integration of [lazysizes](https://github.com/aFarkas/lazysizes) into Shopware 6.

## Installation
* Checkout plugin in `/custom/plugins/SptecLazyLoad`
* Install the plugin with the plugin manager
